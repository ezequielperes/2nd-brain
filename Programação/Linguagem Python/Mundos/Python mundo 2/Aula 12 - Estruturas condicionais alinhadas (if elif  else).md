Condições Alinhadas são [[If]], [[elif]], e [[else]]

[[If]]: -> verifica uma condição
<span style="color: lime;">Bloco</span> [[True]] (executa se a condição for verdadeira)

[[elif]]: -> Ele verifica uma condição e fica **sempre abaixo de um if** ou de um outro elif

[[Else]]: -> executa quando **nenhuma condição anterior** foi verdadeira
<span style="color: red;">Bloco</span> [[False]] (Executa se a condição for falsa)

`if 8 < n  < 10:`
`print('Este número é o 9')`
`elif 1 < n < 3:`
`print('Este número é o 2')`
`elif 4 < n < 6:`
`print('Este número é o 5')`
`else:`
`print('Não sei qual número é este ;-;)`
