# Estudos em Python — 2nd Brain

Repositório de estudos em Python criado para organizar meu aprendizado em lógica de programação
e fundamentos de desenvolvimento de software.

Aqui estão exercícios práticos, anotações e códigos desenvolvidos durante meu estudo contínuo,
com foco em entender a lógica por trás das soluções.

## Conteúdos estudados
- Lógica de programação
- Estruturas condicionais (if / elif / else)
- Estruturas de repetição (for / while)
- Manipulação de listas e tuplas
- Validação de regras e expressões matemáticas
- Escrita de código estruturado e organizado
- Criação e organização de funções (em andamento)

## Organização
- Pastas separadas por aulas, módulos e exercícios
- Códigos em Python (.py) com foco em prática
  Obs: todos os códigos python estão no caminho Programação/Linguagem Python/Exercícios Python
- Anotações complementares para reforçar o aprendizado

para atuar futuramente com desenvolvimento de software, especialmente backend.
