**f-string** é uma forma **mais moderna e direta** de formatação de strings.

Ela permite **colocar variáveis e expressões diretamente dentro do texto**, usando `{}` e colocando `f` antes da string.

Exemplo:

`nome = 'Ezequiel' idade = 18  print(f'Meu nome é {nome} e eu tenho {idade} anos')`

O Python avalia o que está dentro das `{}` **automaticamente**.

Resumindo:  
**f-string é uma string formatada que interpreta variáveis e expressões dentro do texto.**
