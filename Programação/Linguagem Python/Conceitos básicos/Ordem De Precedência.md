
- 1° -> Parênteses ()
- 2° -> [[Potência]] ( ** )
- 3° -> [[Multiplicação]] ( * ), [[Divisão]] ( / ), [[Divisão Inteira]] ( // ) e [[Resto Da Divisão]] (%)

Exemplo De Utilização

2 + 4 ** 2 * 4 + (4 - 2) = 68

Temos que fazer:

1° -> (4 - 2) == 2
2° -> 4 ** 2 == 16
3° -> 16 * 4 == 64
4° -> 2 + 64 + 2 == $68$
