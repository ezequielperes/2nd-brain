Também dá pra se utilizar em textos os operadores básicos

- `+` em texto → concatenação
    
- `*` em texto → repetição
    
- `-`, `/`, `//`, `%`, `**` → **NÃO** funcionam com [[str]] (strings)
- 
Exemplos de Utilização:

----

'Oi' + 'Olá'

Run >

'OiOlá'

----

'Oi' * 3

Run >

'OiOiOi'
